# tree-nodes

Клиент-серверное приложение для создания и манипулирования иерархией псевдо-папок.

Написано с использованием Spring-Boot и ReactJS, а также PrimeReact. 

#### Front-End
Front-end расположен в "src/main/reactfrontend", и представляет из себя функциональный сервер на NodeJS с использованием зависимостей. Spring подхватывает все нужные ресурсы из данной части проекта и использует их как свои.

По пути "src/main/reactfrontend/src" расположен главный файл интерфейса "App.js".

#### Back-End
В серверной части присутствует кнтроллер для обработки запросов и работой с базой данных. И да, всё, что вы изменяете в древе папок, изменяется и в БД. 

В качестве базы данных выбрана PostrgreSQL.
