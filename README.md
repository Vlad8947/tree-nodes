# tree-nodes

Клиент-серверное приложение с использованием Spring-Boot и ReactJS, а также PrimeReact. 

Front-end расположен в src/main/reactfrontend, и представляет изи себя функциональный сервер на NodeJS с использованием зависимостей. Spring подхватывает все нужные ресурсы из данной части проекта и использует их как свои.
